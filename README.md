# Attack Move Unity Template

<img width="100" height="100" alt="logo_100" src="https://github.com/user-attachments/assets/a3661952-908a-4f06-b218-1f41ebe6b024" />

This repository is the template for Attack Move prototypes we build. 
It is designed for the basis of web based
 multiplayer games which we like to build for initial player testing.
To get started, open the **Assets/Scenes/Main.scene**
This has a small example game with a player unit that can be moved around as well as a basic vision system for each team

ParrelSync is included in the project for ease of testing multiplayer games by cloning the Editor
To do this press ParrelSync -> Clones Manager -> Add new clone
Then open this new clone and run both Unity instances to playtest multiplayer on the same machine

Unity version: 6000.0.58f
